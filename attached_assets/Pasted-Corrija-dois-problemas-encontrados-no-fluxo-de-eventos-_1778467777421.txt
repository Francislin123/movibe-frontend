Corrija dois problemas encontrados no fluxo de eventos das Baladas no sistema Movibe.

PROBLEMA 1 — Modal da Balada exibindo quantidade incorreta de eventos

Atualmente existe uma inconsistência no modal/lista de eventos da Balada:

* No banco de dados a balada possui apenas 2 eventos
* Porém no frontend o modal mostra 6 eventos

Investigue e corrija a origem da duplicação.

Validar os seguintes pontos:

1. Verificar se o estado do React está acumulando eventos duplicados
2. Verificar se existe concatenação indevida no setState
3. Garantir que a lista de eventos seja sempre substituída e não concatenada
4. Validar se existe múltipla chamada da API causando duplicidade
5. Garantir que cada evento seja único pelo ID
6. Aplicar deduplicação defensiva utilizando:

```ts
const uniqueEvents = events.filter(
  (event, index, self) =>
    index === self.findIndex((e) => e.id === event.id)
);
```

7. Verificar se o useEffect está disparando múltiplas vezes
8. Verificar dependências do useEffect
9. Garantir que o modal carregue apenas eventos daquela balada específica
10. Validar se o endpoint do backend está retornando eventos duplicados

IMPORTANTE:
A quantidade exibida no frontend deve refletir exatamente o banco de dados.
